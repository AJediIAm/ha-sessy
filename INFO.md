# Sessy Home Assistant integration
Sessy Custom Component for Home Assistant.

Currently supported features:

Sessy Battery
-------------
- Power Status
  - System State
  - Output Power (W)
  - Power Setpoint (W)
  - Renewable Energy Meter
    - Power (W)
	- Current (A)
	- Voltage (mV)

- Network Status
  - WiFi RSSI (dBm)

- Power Strategy
  - Get and set strategy via select entity

- Power Setpoint
  - Get and set via number entity

- Firmware updates

- Configuration (Min/max power)

Sessy P1 Dongle
---------------
- P1 Status
  - Net power flow (kW)

- Network Status
  - WiFi RSSI (dBm)

- Firmware updates
