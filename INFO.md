# Sessy Home Assistant integration
Sessy Custom Component for Home Assistant.

Currently supported features:

Sessy Battery
-------------
- Power Status
  - System State
  - Output Power (W)
  - Power Setpoint (W)
  - Renewable Energy Meter
    - Power (W)
	- Current (A)
	- Voltage (mV)

- Network Status
  - WiFi RSSI (dBm)

Sessy P1 Dongle
---------------
- P1 Status
  - Net power flow (kW)
